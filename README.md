# React Portfolio - Ribadiya Vansh

This is a **React + Vite** migration of the portfolio.

## Getting Started

Since the environment setup was done manually, please follow these steps to run the project:

1.  **Install Dependencies**
    Open your terminal in this folder and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Structure
- `src/components/`: Reusable UI components (Hero, Navbar, etc.)
- `src/App.jsx`: Main application layout
- `src/index.css`: Global styles (migrated from legacy)
- `public/assets/`: Images and static assets
